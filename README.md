#  TrashChute

> *Do whatever you'd like.*


---

## Tools

| Tool | Description |
|------|-------------|
| ✦ Todo | Task list with localStorage persistence |
|  Pomodoro | Focus timer with forest wallpaper, custom wallpaper support, and session tracking |
|  Password Generator | Generates secure passwords with strength indicator and custom options |
|  Markdown Previewer | Live markdown editor with split, editor, and preview modes |
|  Quote Generator | Random advice quotes with save to favorites |
|  Quest Generator | Random activity generator powered by the Bored API — includes an additional mode |
|  Dice & Coin | Roll any dice (D4 to D100) and flip a coin with history tracking |
|  UUID Generator | Generate UUIDs, Short IDs, Nano IDs, and Hex strings in bulk |
|  Image Converter | Convert images between PNG, JPEG, WEBP, BMP and ICO with quality and scale controls |

---

## Stack

- **Next.js 16** — App Router, file-based routing, API routes
- **Framer Motion** — Scroll animations and bento grid transitions
- **HTML5 Canvas** — Custom animated background (fog, binary rain, glitch) and crack effects
- **localStorage** — Client-side persistence for todos and favorite quotes
- **Bored API** — Quest generation

---

## Getting Started

```bash
git clone https://github.com/IMan7736/trashchute
cd trashchute
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

---

## Features

- Custom canvas background with fog blobs, binary rain, and random glitch effects
- Bento grid layout with staggered scroll animations
- Each tool has its own theme and background
- Pomodoro supports custom wallpaper upload
- Quest generator has a Risky Mode with a dark red theme and 500 unique daring quests
- Image converter runs entirely in the browser — no uploads to any server
- Fully client-side — no accounts, no tracking, no nonsense
